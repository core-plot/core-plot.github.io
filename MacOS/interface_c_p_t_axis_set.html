Detailed Description

A container layer for the set of axes for a graph.

Method Documentation

◆ axisForCoordinate:atIndex:

- (nullable CPTAxis *) axisForCoordinate: (CPTCoordinate)  coordinate
atIndex: (NSUInteger)  idx 

Returns the first, second, third, etc. axis with the given coordinate value.

For example, to find the second x-axis, use a coordinate of CPTCoordinateX and idx of 1.

Parameters
coordinateThe axis coordinate.
idxThe zero-based index.
Returns
The axis matching the given coordinate and index, or nil if no match is found.

implementation

Initializes a newly allocated CPTAxisSet object with the provided frame rectangle.

This is the designated initializer. The initialized layer will have the following properties:

  • axes = empty array
  • borderLineStyle = nil
  • needsDisplayOnBoundsChange = YES

atPoint: (CGPoint)  interactionPoint 
implementation

Informs the receiver that the user has pressed the mouse button.

The event will be passed to each axis belonging to the receiver in turn. This method returns YES if any of its axes handle the event.

Parameters
eventThe OS event.
interactionPointThe coordinates of the interaction.
Returns
Whether the event was handled or not.
