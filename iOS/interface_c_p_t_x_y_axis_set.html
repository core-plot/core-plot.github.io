Detailed Description

A set of cartesian (X-Y) axes.

Initializes a newly allocated CPTXYAxisSet object with the provided frame rectangle.

This is the designated initializer. The axes array will contain two new axes with the following properties:

◆ layoutSublayers()

- (void) layoutSublayers
implementation

Updates the layout of all sublayers. Sublayers (the axes) fill the plot area frame’s bounds.

This is where we do our custom replacement for the Mac-only layout manager and autoresizing mask. Subclasses should override this method to provide a different layout of their own sublayers.

Reimplemented from CPTLayer.
